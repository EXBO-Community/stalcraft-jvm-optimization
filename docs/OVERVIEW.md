# Техническая информация

## Механизм работы

Враппер использует механизм IFEO (Image File Execution Options) для перехвата запуска игры.
Когда запускается `stalcraft.exe` или `stalcraftw.exe`, вызов перенаправляется на враппер. Его поведение:

1. Загружает активный конфигурационный файл из каталога `configs/`.
2. Убирает конфликтующие флаги из аргументов оригинального лаунчера.
3. Создаёт процесс напрямую через `ntdll!NtCreateUserProcess`, чтобы избежать повторного перехвата через IFEO.
4. Поднимает приоритеты памяти и ввода/вывода с помощью `NtSetInformationProcess`.
5. Завершается после того, как у игрового процесса появится первое видимое окно.

## Взаимодействие через CLI

Установка перехвата IFEO

```bash
wrapper.exe --install     # установить перехват IFEO
```

Проверка статуса перехвата

```bash
wrapper.exe --status      # проверить статус перехвата
```

Удаление перехвата IFEO

```bash
wrapper.exe --uninstall   # удалить перехват IFEO
```

## Сборка проекта

`wrapper.exe` можно не только скачать из релизов, но и собрать самому.
Для этого достаточно выполнить команду:

```bash
cd wrapper & go build -o wrapper.exe -ldflags="-s -w" .
```
